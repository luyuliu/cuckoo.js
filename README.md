# cuckoo.js

An Javascript library to implement cuckoo search.

## Import
Web browser environment: 
```HTML
<script src="cuckoo.js"></script>
```

Node.js environment (not available yet):
```bash
$ npm install cuckoo
```

## Usage
```js
var cuckoo = new Cuckoo(nestCount, dimension, Pa, Lb, Ub);
```
